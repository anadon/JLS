Keyboard-only Construction

You can lay out and wire a circuit entirely from the keyboard, with no pointing device. The editing area behaves like any other component: give it keyboard focus by selecting its tab (or clicking in it once), and the keys below then drive the whole construction.

The caret

The arrow keys move a small crosshair called the caret around the drawing grid. The caret is the keyboard's equivalent of the mouse pointer: it marks where the next element will drop and where a wire will start or end. It always sits on a grid line, so wires you start or finish at the caret line up exactly with the ports on gates and pins.

Placing an element

  1. Choose an element from the Element menu on the tool bar (it is fully keyboard navigable) and answer its creation dialog as usual.
  2. The new element appears attached to the keyboard. Use the arrow keys to move it one grid step at a time to where you want it.
  3. Press Enter to drop it. If a message warns of an overlap, move it clear first; Enter has no effect while it overlaps something.

Moving something already placed

With nothing being placed, move the caret over an element and press Enter to select it. The arrow keys then nudge the selected element one grid step at a time (this is undoable, like a mouse drag). Press Enter again to finish and deselect.

Wiring port to port

To connect two elements with a wire:

  1. Move the caret onto the source port (for example a gate's output).
  2. Press w to start a wire there.
  3. Press Enter to anchor the wire to that port.
  4. Use the arrow keys to run the wire's free end across to the destination port (a gate input or a pin).
  5. Press Enter on the destination port to complete the connection. Press Esc at any time to abandon the wire.

Because the caret and every port share the grid, ending a wire on a port connects them exactly, the same way touching two ports with the mouse does.

Rotating, flipping, and the rest

The plain keys r (rotate clockwise), shift-r (rotate counter-clockwise), and f (flip) act on the selected element, and every other operation on the Keyboard Accelerators page works from the keyboard too. When the circuit is built, press F5 to simulate it.

A worked example: A + ~B

This is the first circuit in the tutorial, built with no mouse:

  1. Choose an OR gate, move it to the center with the arrow keys, and press Enter.
  2. Choose a NOT gate, move it with the arrow keys so its output is a little to the left of one of the OR gate's inputs, and press Enter.
  3. Move the caret onto the NOT gate's output, press w, press Enter, move the caret across to the OR gate's input, and press Enter to wire them.

The two gates are now the A + ~B skeleton, ready for input and output pins and simulation.
