Keyboard Accelerators (Hot Keys)

Many mouse functions can be performed by keyboard accelerators (hot keys). The keys and their functions are:

Function Key
Select All ctrl-a
Cut ctrl-x
Copy ctrl-c
Paste ctrl-v
Delete delete
Undo ctrl-z
Redo ctrl-y (shift-cmd-z on macOS; cmd-y also works)
Start Wire w
End Wire esc
View Element Contents v
Rotate Element Clockwise r
Rotate Element Counter-Clockwise shift-r
Flip Element f
Watch/Unwatch Element ctrl-w
Add/Remove Probe ctrl-p
Modify Element ctrl-m
Change Timing ctrl-t
Lock Element ctrl-l
Move Caret / Following Element / Wire End / Selection arrow keys
Place, Wire, or Select (keyboard commit) enter

Many keys only work when the cursor is in a certain place (e.g., on an element, on a wire, in "empty" space) - otherwise they have no effect. Start Wire and Watch/Unwatch used to share ctrl-w, which put two functions on one stroke. They are now separate: plain w starts a wire, and ctrl-w (cmd-w on macOS) toggles watch on a single selected element.

The editing area holds keyboard focus like any other component: click in it (or just select its tab) and the hot keys above work until you click somewhere else. Focus no longer follows the mouse pointer. The editing operations also appear in the menu bar's Edit and Element menus, each showing its accelerator; those Ctrl/Cmd accelerators work no matter which part of the window has focus. File operations (New, Open, Save, Save As, Exit) likewise have their own menu accelerators, shown in the File menu.

The arrow keys and Enter make it possible to build a circuit with no pointing device at all. See Keyboard-only Construction for the full workflow.
