Creating a Shift Register

A shift register is a combinational shifter: its output is its data input shifted by the value on its amount input. It holds no state and has no clock.

Set the number of bits of the data input (at least 2) by entering a number or using the keypad. The number of bits of the amount input will be automatically set to the ceiling of the log2 of this number (e.g., 8 bits will result in a 3-bit amount input).

The shift type dictates the direction of the shift and how the vacated bits are filled: Shift Left and Shift Right fill with 0; Shift Right Arithmetic fills with a copy of the sign (highest) bit. Shifting by an amount greater than or equal to the number of bits outputs 0 (or all sign bits, for an arithmetic right shift).

Click on OK to place the element, or Cancel to cancel its creation.
